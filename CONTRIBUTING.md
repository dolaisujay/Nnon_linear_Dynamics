# Contributing

Thanks for your interest in contributing!

## Ways to help

- Report bugs and usability issues (include MATLAB release + toolboxes used).
- Propose improvements to documentation and examples.
- Contribute analysis scripts, verification cases, or improvements to plots/exports.

## Filing an issue

Please include:

- MATLAB version (e.g., R2022a) and installed toolboxes (Simulink/Simscape/etc.).
- OS version.
- Steps to reproduce (minimal parameter set if possible).
- Expected vs actual behavior.
- Any error text / stack trace.

## Development notes

- Keep MATLAB code compatible with **R2022a** unless there is a strong reason to raise the minimum.
- Prefer small, focused changes per PR.
- If you move files, update `README.md` and any path assumptions in code.
